# Bamazon

### About

This is a Node.js app that utilizes MySQL to create a basic command line based storefront. Utlizing - 
    * Inquirer
    * MySQL

In order to run this application, you will need a MySQL database. You will create a *bamazon* database and *products* table with the SQL code found in the associated files. 

**Application**

The app will take in orders from customers and deplete stock from store inventory. The app will display an inventory list of items a customer may buy from. The app will prompt the user to enter an item id then quantity for purchase. The database will be updated to reflect the new inventory or alert the user that their request cannot be fulfilled due to insufficient inventory. 
With a successful purchase the app will display the total cost of the purchase to the user.
